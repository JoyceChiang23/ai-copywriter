# AI 文案生成器 - 推廣策略與實施計畫

## 工具已完成 ✅

檔案名稱：`ai-copywriter.html`

**功能列表：**
- ✅ IG 貼文文案（引發互動）
- ✅ Reels 腳本（15-30 秒）
- ✅ 限動文案（簡短吸睛）
- ✅ 企劃書大綱（5 頁內容）
- ✅ 商品描述（蝦皮、momo、PChome、Shopee、官網）
- ✅ 行銷文案
- 目標客群選擇（年輕女性、上班族、創業者、居家愛好者、一般大眾）
- 語氣風格選擇（專業、親切、幽默、勵志、輕鬆）

---

## 使用說明

### 第 1 步：打開網頁
1. 在檔案管理員中，進入 `Claude_Project` 資料夾
2. 雙擊 `ai-copywriter.html`
3. 在瀏覽器中打開

### 第 2 步：設定 API Key（一次性設定）
1. 去 https://platform.openai.com/account/api-keys 複製你的 API Key
2. 第一次使用工具時，會跳出視窗要求輸入 API Key
3. 貼上後，工具就會開始生成文案

**費用說明：**
- OpenAI API 按使用量計費（約 NT$0.01-0.1 元/次）
- 免費試用額度通常有 $18 美金

### 第 3 步：選擇並生成
1. 選擇文案類型（IG 貼文、Reels、限動、企劃、商品、行銷）
2. 輸入需求（越詳細越好）
3. 選擇目標客群和語氣
4. 點擊「生成文案」
5. 複製結果使用

## 常見問題

**Q: 沒有 API Key 怎麼辦？**
A: 去 https://platform.openai.com/account/api-keys 註冊免費帳號，系統會給你 $18 美金試用額度（約 2-3 個月用量）。

**Q: 文案品質不好怎麼辦？**
A: 調整「需求描述」要更詳細，「目標客群」和「語氣」要更明確。好的 input = 好的 output。

**Q: 怎麼知道朋友有沒有轉發？**
A: 可以用 bit.ly 或 pse.is 製作短連結追蹤（但這步可以先跳過，等有真正流量再做）。

**Q: 什麼時候推廣給陌生人？**
A: 先靠口碑傳播 2-3 週，如果效果好再考慮。朋友圈推薦比陌生廣告更有效。
